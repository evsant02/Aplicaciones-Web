Planificación

Primera entrega

Para la planificacion del trabajo grupal, en esta primera fase hemos tomado las siguientes decisiones:

  • Hemos descrito como tareas el desarrollo de cada una de las páginas indicadas en el enunciado.
  • Luego para la división de las páginas a crear entre cada uno de los integrantes, hemos repartido por selcción cada una de ellas.
  • La página sobrante le fue asignada a Umaima debido a que ella también se encargaba de Índice cuya descripcion corta ya teníamos.
  • Para esta primera fase del proyecto hemos decidido tener una reunión de control descrita en la tabla de organización.

Entregas sucesivas

Para las siguientes fases de la práctica hemos cambiado algunos aspectos de la organización

  • Las tareas a desarrollar pasan a ser las diferentes funcionalidades acordadas.
  • Las funcionalidades se han repartido de manera aletoria.
  • En la fase 2 las funcionalidades a implementar son Alta de perfiles, Mi perfil y Administrar actividades, para la fase 3 se implementarán Apuntarse a actividad, Consulta y Donaciones.
  • Hemos decidido tener más reuniones de control de progreso al ser fases con mayor complicación.
